# 添加文件

angle.c,angle.h

* 新增两个函数 

  * servo_angle2wheel_angle（）:舵机转向角度与前轮角度呈线性关系，这里进行了转换，调试时修改k这个系数使二者转换关系更为准确

  * speed_rate（）：前提假设小车转向时右后轮始终保持匀速，左轮根据角度进行调整。该函数依据此假设计算出二者的比例关系。
*  函数的使用

   * 主函数中获取舵机判断误差后的转向角度后，使用函数一得到轮子的转向角，此后在中断中计算左轮新的目标速度完成差速辅助转向。

# 调试说明
### 需要获取的数据 

车轴距：CAR_LENGTH，前后轮车轴之间的距离（在angle.c文件中设置）

车后轮距：WHEELBASE，车后轮的间距（在angle.c文件中设置）

### 注意事项
中断中需明确编码器12（encoder_1/encoder_2），电机12（motor_1/motor_2）的对应关系，不要搞反了，左电机的线接在电机驱动右侧，右电机的线接在电机驱动左侧
